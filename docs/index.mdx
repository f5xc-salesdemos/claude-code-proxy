## Why Claude Code Proxy?

Claude Code is a powerful AI coding assistant, but it only connects to
Anthropic's API. This proxy translates Claude API requests into
OpenAI-compatible calls so you can use **any provider** you want.

<CardGrid stagger>
  <Card title="Any Provider" icon="rocket">
    Connect to OpenAI, Azure OpenAI, Ollama, or any OpenAI-compatible API.
  </Card>
  <Card title="Drop-in Replacement" icon="puzzle">
    Set one environment variable and Claude Code works exactly as before.
  </Card>
  <Card title="Model Mapping" icon="setting">
    Map Claude model tiers (haiku, sonnet, opus) to any upstream model.
  </Card>
  <Card title="Full Feature Support" icon="approve-check">
    Streaming, function calling, image inputs, and web search all work
    out of the box.
  </Card>
</CardGrid>
