# Claude Code Proxy

Use Claude Code with any OpenAI-compatible provider.

## Why Claude Code Proxy?

Claude Code is a powerful AI coding assistant, but it only connects to
Anthropic's API. This proxy translates Claude API requests into
OpenAI-compatible calls so you can use **any provider** you want.

- **Any Provider** -- Connect to OpenAI, Azure OpenAI, Ollama, or any OpenAI-compatible API.
- **Drop-in Replacement** -- Set one environment variable and Claude Code works exactly as before.
- **Model Mapping** -- Map Claude model tiers (haiku, sonnet, opus) to any upstream model.
- **Full Feature Support** -- Streaming, function calling, image inputs, and web search all work out of the box.
